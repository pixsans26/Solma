# Features, Pages, and Routes

## Authentication
- **Login** (`/login`): User login page
- **Signup** (`/signup`): New user registration
- **OTP Verification** (`/otp`): One-time password entry for authentication

## Content Pages
- **Home** (`/`): Main landing page with featured content
- **Movies** (`/movies`): Browse all movies
- **Web Series** (`/series`): Browse all web series
- **Short Films** (`/short-films`): Browse short films
- **Songs** (`/songs`): Browse songs
- **Music Videos** (`/music-videos`): Browse music videos
- **Movie Details** (`/movies/:id`): Details, watch now, trailer, IMDB ratings, rent/buy options
- **Series Details** (`/series/:id`): Details, episodes, watch now, trailer, IMDB ratings

## User & Account
- **Profile** (`/profile`): User profile, watchlist, downloads, subscriptions
- **Settings** (`/settings`): App and account settings
- **App Settings** (`/app-settings`): Language, preferences, etc.
- **Subscriptions** (`/subscriptions`): Manage and purchase subscriptions
- **Downloads** (`/downloads`): In-app downloaded content

## Features
- **Multiple Languages**: UI and content language selection
- **IMDB Ratings**: Show ratings on details pages
- **Watch Now**: Direct play option for movies/series
- **Trailer Option**: Watch trailer before full content
- **Movie Rent/Buy**: Rent or buy movies
- **In-App Download**: Download content for offline viewing
