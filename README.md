# OTT Streaming App Prototype (Mobile-First)

A modern, mobile-first OTT streaming prototype built with Vite + React.

## Features
- Fixed 390px width, centered on desktop
- Dark Netflix-inspired theme, compact UI
- Sticky bottom navigation (Home, Music, Search, Download, Profile)
- Reusable MovieCard and MusicCard components
- Minimal custom video player
- Framer Motion transitions
- Zustand for state management
- Static JSON for movies/music
- Optimized for mobile UX and performance

## Getting Started
```sh
npm install
npm run dev
```
